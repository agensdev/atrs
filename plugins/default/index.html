Default

The default plugin will allow you to set a fallback value if no attribute or nil is given when initializing. The option default is made available for your attributes and supports a literal value, a proc/lambda or a :some_method_name_ending_with_default symbol as input.

Example

class Foo
  include Wardrobe
  plugin :default

  attribute :number, Integer, default: 10
  attribute :name,   String, default: -> { 'No Name' }
  attribute :status, Symbol, default: :new
  attribute :count,  Integer, default: :count_default

  def count_default
    number * 2
  end
end

Foo.new
#=> #<Foo:0x007f8953813718 @number=10, @name="No Name", @status=:new, @count=20>
