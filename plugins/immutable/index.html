Immutable

The immutable plugin will make any Wardrobe model immutable and expose a #mutate method that will return a new object.

Example

class User
  include Wardrobe
  plugin :immutable
  attribute :foo, String
  attribute :bar, Array[String]
end

user = User.new(foo: 'Test', bar: ['Test 1'] )

user.foo = 'Changed' #=> NoMethodError: undefined method `foo='...
user.bar << 'Test 2' #=> RuntimeError: can't modify frozen...

updated_user = user.mutate do |user|
  user.foo = 'Changed'
  user.bar << 'Test 2'
end

user #=> <User:0x007fb83a879c78 @foo="Test", @bar=["Test 1"]>
updated_user #=> <User:0x007fb838246268 @foo="Changed", @bar=["Test 1", "Test 2"]>
