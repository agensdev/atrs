Nil if Empty

The nil_if_empty plugin works with String, Array and Hash and will set the value to nil when coercing if the value returns true when #empty? is called.

Example

class Foo
  include Wardrobe
  plugin :nil_if_empty
  attribute :string, String, nil_if_empty: true
  attribute :array, Array, nil_if_empty: true
  attribute :hash, Hash, nil_if_empty: true
end

foo = Foo.new(string: '', array: [], hash: {})
foo #=> <Foo:0x007fb8389a2610 @string=nil, @array=nil, @hash=nil>
