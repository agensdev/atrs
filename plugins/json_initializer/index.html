JSON Initializer

This plugin will allow you to initialize Wardrobe models with a serialized json string. The plugin will check if MultiJson is available or fallback to using JSON.

Example

class Foo
  include Wardrobe
  plugin :json_initializer

  attribute :bar, String
end

Foo.new('{"bar":"value"}')
#=> <Foo:0x007fa1779e1478 @bar="value">
